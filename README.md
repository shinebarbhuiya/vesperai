## About The Project



VesperAI is a fun project born out of the idea to create a free, unlimited AI image generator using Next.js. The inspiration for this project came to me while exploring services that offer API endpoints for text-to-image generators. During my search, I stumbled upon [Deepinfra](https://deepinfra.com/), a platform providing free generations for some of their Language Model Models (LLMs) and text-to-image models.

## How It Works

After discovering that I could generate unlimited images without the need for authentication, I decided to delve deeper. By inspecting the network requests made by the Deepinfra website, I was able to replicate those API calls using Next.js. The result is VesperAI, an AI image generator that allows users to create images directly from their browsers.

### Key Points:

- **No Login Required**: Users can generate images without the need for any authentication or login.
- **Unlimited Generation**: Experience unlimited image generation, making it a playful and enjoyable tool.

## Technical Details

VesperAI utilizes Next.js to make API calls to Deepinfra for image generation. The API calls are made directly from the client (user's browser and IP), ensuring that there are no rate limits imposed, as requests are distributed across various IP addresses.
